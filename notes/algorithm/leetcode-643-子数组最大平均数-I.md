# LeetCode 643. 子数组最大平均数 I

📅 2026年6月24日 | 🏷️ 滑动窗口 · 定长窗口 | 灵神题单 ② | 难度：简单

---

## 1. 题意拆解

> 给整数数组 `nums` 和整数 `k`，找长度为 k 的**连续子数组**，使它的平均值最大。

**大白话**：和 1456 几乎一样，把「数元音」换成「求和」，最后除以 k。

```
nums = [1, 12, -5, -6, 50, 3], k = 4

长度4的子数组：
[1,12,-5,-6]  → 和=2   → 平均=0.5
[12,-5,-6,50] → 和=51  → 平均=12.75  ← 最大！
[-5,-6,50,3]  → 和=42  → 平均=10.5

答案：12.75
```

---

## 2. 思路递进

### 暴力（❌）
枚举每个子数组，求和算平均 → O(n×k)

### 滑动窗口（✅）
窗口滑一格，新元素加进来、旧元素减出去 → O(n)

⚠️ **关键优化**：求「最大平均」等价于「最大和」，最后再除以 k，避免循环内做浮点除法。

---

## 3. 完整代码

```java
class Solution {
    public double findMaxAverage(int[] nums, int k) {
        int n = nums.length;
        int sum = 0;      // 当前窗口和
        int maxSum = 0;   // 历史最大和

        for (int i = 0; i < n; i++) {
            // ===== ① 入窗 =====
            sum += nums[i];

            // ===== ② 更新答案 =====
            if (i == k - 1) {
                // 第一个窗口，直接作为初始最大值
                maxSum = sum;
            } else if (i >= k) {
                // 后续窗口，比较取最大
                maxSum = Math.max(maxSum, sum);
            }

            // ===== ③ 出窗 =====
            if (i >= k - 1) {
                sum -= nums[i - k + 1]; // 移出左边界元素
            }
        }

        // ⚠️ 最后统一除，避免精度丢失
        return (double) maxSum / k;
    }
}
```

---

## 4. 逐行拆解 + 易错点

```java
if (i == k - 1) {
    maxSum = sum;   // 第一个窗口先给初始值
} else if (i >= k) {
    maxSum = Math.max(maxSum, sum); // 后续窗口才比较
}
```
⚠️ **为什么分两种情况？** 如果直接 `Math.max(maxSum, sum)`，maxSum 初值是 0，遇到负数窗口（如 `[-1,-2,-3]`）会错。用 `i == k-1` 初始化更安全。

```java
sum -= nums[i - k + 1];
// ⚠️ 左边界 = i - k + 1
// i=3, k=4 → 减去 nums[0]，新窗口 [1,2,3,4]
```

```java
return (double) maxSum / k;
// ⚠️ 最后才除，别在循环里每次算 (double)sum/k
// 循环里除：每次浮点运算 + 精度误差
```

## 6. 考点总结

| 考点 | 说明 |
|------|------|
| 定长滑窗模板 | 入→更新→出，和 1456 完全一致 |
| 整数→浮点 | 循环内用 int 算和，最后转 double 除 |
| maxSum 初始化 | 用第一个窗口初始化，避免全负数出错 |
| 同类题 | 1456（数元音）、1052（+base 预处理） |

| 入窗条件 | 元音才+1 | 全加 | 生气才+1 |
| 出窗条件 | 元音才-1 | 全减 | 生气才-1 |
