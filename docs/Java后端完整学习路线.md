### 模块一：Java SE 核心（4-5周）

**第1章 基础语法（1周）**
- 变量与数据类型、运算符、流程控制
- 数组（遍历/排序/二分查找/Arrays工具类）
- 方法（定义/重载/可变参数/递归）⭐

**第2章 面向对象（1.5周）⭐⭐⭐**
- 类与对象、封装（private/getter/setter）
- 继承（extends/super/方法重写/final）⭐
- 多态（向上转型/向下转型/instanceof）⭐
- 抽象类 vs 接口 ⭐ 面试高频
- 内部类、枚举 enum

**第3章 集合框架（1周）⭐⭐⭐**
- ArrayList 动态扩容 / LinkedList / 对比
- HashMap 底层（数组+链表+红黑树）⭐⭐⭐
- HashMap 扩容（为什么容量是2的幂？）⭐⭐
- ConcurrentHashMap（jdk7分段锁→jdk8 CAS+synchronized）
- HashSet/TreeSet、Comparable vs Comparator

**第4章 异常 + 泛型 + IO（0.5周）**
- 异常体系、try-catch-finally执行顺序 ⭐
- 泛型类型擦除 ⭐
- IO流基础、序列化概念

**第5章 JDK 8+ 新特性（0.5周）**
- Lambda表达式 / Stream API
- Optional / 方法引用
- JDK 17/21 关键新特性（record/sealed class/virtual thread概念）
